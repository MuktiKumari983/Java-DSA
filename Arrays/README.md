# Arrays

## Overview

Arrays are one of the most fundamental data structures in Data Structures and Algorithms (DSA).

This folder contains array-based problems solved in Java along with different problem-solving patterns commonly used in coding interviews.

---

## Concepts Covered

- Array Traversal
- 2D Array Traversal
- Maximum Subarray (Kadane's Algorithm)
- Running Minimum
- Sorting & Comparison
- Frequency Counting
- HashSet for Distinct Elements
- Interval Overlap
- Monotonic Arrays

---

## Problems Solved

| Problem | Pattern |
|----------|----------|
| Richest Customer Wealth | 2D Array Traversal |
| Best Time to Buy and Sell Stock | Running Minimum |
| Maximum Subarray | Kadane's Algorithm |
| Height Checker | Sorting + Comparison |
| Monotonic Array | Flag-Based Traversal |
| Teemo Attacking | Interval Overlap |
| Majority Element | Frequency Counting |
| Third Maximum Number | HashSet |

---

## Time Complexity Patterns

### Traversal
- Time: O(n)
- Space: O(1)

### Sorting
- Time: O(n log n)
- Space: O(1) / O(n)

### HashSet
- Insertion: O(1)
- Lookup: O(1)

### Kadane's Algorithm
- Time: O(n)
- Space: O(1)
